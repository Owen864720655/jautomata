Shortest Distances and String weights

Common uses of automata are computing the weight of a string, computing the n shortest paths, and computing the n best strings. All of these can be computed with generic shortest distance algorithms. In this case generic applies to the semiring. In this section, you can find how to do this.

Using the following code, we can load an unweighted automaton from a file:
  1. File file = new File("filename");
  2. InputStreamReader isr = new InputStreamReader(new FileInputStream(file, "UTF-8"));
  3. Automaton<Character, Boolean> acceptor = Automata.readUnweighted(isr, "att");
